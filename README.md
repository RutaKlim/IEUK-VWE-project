# IEUK work experience engineering project 2026

My task was to process 24 hours worth of data, using python pandas, that reports which wind turbines fail, using their stats.

As an introduction, I never used Python, Pandas or Dockerfile before this project. We were given two topics, one was based on software developing, and this one on data science. I am on the basic computer science course and I have experience with the first project, so I feel like I would not have learnt anything from this event, so I decided to do the 'engineering' task as a challenge and I ended up learning a lot more than I would not have learnt from the other task.
